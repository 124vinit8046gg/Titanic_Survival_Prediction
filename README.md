# Titanic Survival Prediction

A machine learning project that predicts passenger survival on the Titanic using Logistic Regression.

## Features

- Data preprocessing (handling missing values, encoding categorical features)
- Visualization of survival patterns by gender
- Logistic Regression model training
- Model evaluation with accuracy, confusion matrix, and classification report

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python Titanic_Survival_Pridiction.py
```

## Dataset

The project uses the Titanic dataset from [Data Science Dojo](https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv).

## Results

The model provides:
- Accuracy score
- Confusion matrix
- Detailed classification report

## Dependencies

- pandas
- seaborn
- matplotlib
- scikit-learn
